# Sure Diagnostics — Partners Hub

Partner-facing portal for referring doctors and clinics.

**Live URL:** https://sure-partners.netlify.app

## Usage
Partners log in with their Partner ID and portal password to view:
- Referral dashboard and KPIs
- Patient records and test results
- Rebate statements
- Branch-level analytics

## Deployment
Connected to Netlify. Push to `main` branch → auto-deploys in ~30 seconds.
